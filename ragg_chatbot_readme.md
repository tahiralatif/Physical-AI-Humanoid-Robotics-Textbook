# RAG Chatbot Feature

This branch contains the implementation of the RAG (Retrieval-Augmented Generation) chatbot functionality for the Physical AI & Humanoid Robotics Textbook project.

## Features

- FastAPI backend with chat endpoints
- Qdrant vector database integration for document retrieval
- OpenAI integration for chat completions
- Next.js frontend with chat interface
- Authentication system with Better Auth
- Document ingestion pipeline

## Files

This branch includes all the necessary files for the RAG chatbot functionality including:
- FastAPI backend in `fastapi/` directory
- Next.js frontend in `nextjs/` directory
- Docusaurus documentation in `docusaurus/` directory
- Configuration and setup files
