# Notation Guide

## Mathematical Notation

- **Vectors**: Bold lowercase v
- **Matrices**: Bold uppercase M
- **Scalars**: Italic x

## ROS 2 Notation

- **Nodes**: `node_name`
- **Topics**: `/topic_name`
- **Parameters**: `~parameter_name`

## Code Conventions

- Python: PEP 8
- C++: Google Style Guide
