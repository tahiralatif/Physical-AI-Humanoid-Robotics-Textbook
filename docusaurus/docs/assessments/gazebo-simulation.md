# Assessment: Create a Robot Simulation

**Deadline**: End of Week 7  
**Weight**: 20%

## Objective

Model and simulate a custom robot in Gazebo.

## Requirements

1. **URDF Model**: Mobile robot with 2 wheels + caster
2. **Sensors**: Camera + LIDAR
3. **Gazebo World**: Obstacles and targets
4. **ROS 2 Integration**: Control via `cmd_vel`

Submit URDF files + simulation demo.
