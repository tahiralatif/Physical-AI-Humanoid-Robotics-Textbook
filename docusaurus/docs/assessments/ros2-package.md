# Assessment: Build a ROS 2 Package

**Deadline**: End of Week 5  
**Weight**: 20% of course grade

## Objective

Create a complete ROS 2 package demonstrating node creation, topic communication, services, parameters, and launch files.

## Requirements

Build a **robot teleoperation system** with:

1. **Joystick Node** - Publishes joystick input
2. **Velocity Controller** - Subscribes to joystick, publishes velocity commands
3. **Service**: Emergency stop
4. **Parameters**: Max speed, acceleration limits
5. **Launch file**: Start all nodes

## Rubric

- Code quality: 30%
- Functionality: 40%
- Documentation: 20%
- Testing: 10%

Submit code + 2-minute demo video.
