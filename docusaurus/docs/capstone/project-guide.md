# Capstone Project: Autonomous Humanoid Robot

## Project Overview

Build an autonomous humanoid robot system that demonstrates all course concepts:

- **ROS 2**: Multi-node architecture
- **Simulation**: Gazebo or Isaac Sim
- **Perception**: Object detection and tracking
- **VLA**: Natural language command processing
- **Humanoid locomotion**: Walking and navigation

## Requirements

### Core Functionality

1. **Natural Language Interface**
   - Accept commands like "Go to the kitchen and pick up the red mug"
   - Parse intent using VLA model

2. **Autonomous Navigation**
   - SLAM-based mapping
   - Path planning around obstacles
   - Dynamic obstacle avoidance

3. **Object Manipulation**
   - Detect objects using vision
   - Plan grasp approach
   - Pick and place objects

4. **Bipedal Locomotion**
   - Walk on flat surfaces
   - Maintain balance
   - Recover from small disturbances

## Deliverables

1. **Code Repository**: Complete ROS 2 workspace
2. **Documentation**: README with setup instructions
3. **Demo Video**: 5-minute demonstration
4. **Technical Report**: Architecture and design decisions

## Evaluation Criteria

| Category | Weight | Criteria |
|----------|--------|----------|
| System Integration | 30% | All components work together |
| Code Quality | 20% | Clean, documented, modular |
| Innovation | 20% | Creative solutions |
| Demonstration | 30% | Successful task completion |

## Suggested Timeline

- **Week 13 Day 1-2**: Design architecture
- **Day 3-4**: Integrate components
- **Day 5-6**: Testing and debugging
