# Detailed Course Structure

## 13-Week Breakdown

| Week | Module | Topic | Key Concepts |
|------|---------|-------|--------------|
| 1-2 | Intro | Physical AI Foundations | Course overview, robotics stack |
| 3 | ROS 2 | Fundamentals | Nodes, topics, services |
| 4 | ROS 2 | Communication | Pub/sub, action servers |
| 5 | ROS 2 | Advanced | Parameters, launch files |
| 6 | Digital Twin | Gazebo Basics | Simulation, world files |
| 7 | Digital Twin | URDF & Sim | Robot modeling, physics |
| 8 | Isaac | Isaac Sim Intro | NVIDIA Omniverse |
| 9 | Isaac | Perception | Computer vision pipelines |
| 10 | Isaac | Manipulation | Motion planning, grasping |
| 11 | VLA | Vision-Language-Action | Multimodal AI models |
| 12 | Humanoids | Humanoid Basics | Bipedal locomotion |
| 13 | Integration | Full System | Capstone project |

## Weekly Time Commitment

- **Reading**: 2-3 hours
- **Coding exercises**: 3-4 hours
- **Lab assignments**: 2-3 hours
- **Total**: 8-10 hours per week

## Assessment Structure

1. **ROS 2 Package** (Week 5): Build a complete ROS 2 application
2. **Gazebo Simulation** (Week 7): Create a robot simulation
3. **Isaac Perception** (Week 10): Implement vision pipeline
4. **Capstone Project** (Week 13): Autonomous humanoid system

## Capstone Project

Build an **autonomous humanoid robot** that can:

- Navigate an environment using SLAM
- Detect and pick up objects using vision
- Respond to natural language commands
- Execute multi-step tasks

## Resources Provided

- **Code templates**: Starter code for all exercises
- **Dataset links**: Pre-collected sensor data
- **Video tutorials**: Supplementary explanations
- **Office hours**: Weekly Q&A sessions
